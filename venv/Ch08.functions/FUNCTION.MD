        # 외부 함수의 변수를 참조해야한다.
        # 외부 함수가 내부 함수를 반환해야 한다.
        
* 이터레이터
    - 이터러블 객체란 무엇일까?

    - 문자열, 리스트, 튜플, 딕셔너리, range 객체    

* 제너레이터
    - 제너레이터란?
        # 이터레이터를 만드는 함수 , 좀 더 간편하게 만드는 방법

        # 함수안에 yeild를 사용한다. (return 과 비교)
        # 제너레이터 표현식을 사용할 수 있다. (좀 더 간편하게 사용할 수 있다.)
        # 메모리 사용히 효율적이다.
    - return 과의 차이점
        # return 작업을 하던 도중 만나면 함수실행이 끝남 
        # yield는 작업별로 나눌 수 있다.
* 데코레이터
    - 데코레이터란?

        # 주로 로깅, 권한에 사용 
    - 데코레이터 사용방법
