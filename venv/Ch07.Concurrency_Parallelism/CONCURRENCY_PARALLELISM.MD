# 동시성과 병렬성
```buildoutcfg
* 용어 개념 정리 (프로그램, 프로세스, 스레드)
* 동시성 프로그래밍
* 병렬성 프로그래밍

# 프로그램 특징
    - 작업을 수행하는 명령어 집합
    - ex) 파이썬, 엑셀, 파워포인트
    
# 프로세스 특징
    - 실행중인 프로그램    
    
# 스레드 특징
    - 프로세스에서 실행되는 작업
    - 프로세스는 기본적으로 하나의 스레드로 구성
    - 경우에 따라 여러개의 스레드로 구성이 가능하다(=>멀티스레딩)
    
# 멀티 스레딩 (동시성)
    - 스레드 교체가 빠르게 일어나므로 거의 동시에 일어나는 것처럼 느끼는 방식
    
# 멀티프로세싱 (병렬성)
    - 프로세스 자체가 여러 개 있으므로, 스레드 교체가 일어나지 않고 진짜로 동시에 작업이 진행됨.  
```
